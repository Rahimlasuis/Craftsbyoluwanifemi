Crafts_by_oluwanifemi

This project is a mini but standard e-commerce sales website.
It built using basically HTML, CSS and Javascript
This application allows user to dynamically scroll between different crochet product cards categorized into three different sections which are Top products, Limited deals and Pre-Order categories.
Each product cards contain the product image, the cost price and an add to cart botton.
User can add any item in multiples as much as they want and later view their cart by clicking on the cart botton at the top of the page. Where the total amount of all the items added to cart is displayed.
There is also an accout option. User are requires to create an account with us so we can know who you are and also have some of your details like your phone number, gmail and also house adress.
Due to the fact that this projects was created for an hackclub event and not for a proper real-life use there is no proper checkout property but there is still a checkout button inside the cart pop-up.
This is a frontend appllication with no heavy framework or external database application
Some of the information I needed in this project was saved on Wen Storage API ("localStorage")
There is also an help modal pop-up in at the top of the with about two or three basic instructions for some basic actions.

Other features used include Mozilla Developer Network, Javascript HTML DOM Elements, stack Overflow
